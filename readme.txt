	IDEA的target目录下面会生成liuxv.war包，将这个自动打的包部署到与服务器上面。
	云服务器的配置环境如下：tomcat9.0 ，java8，MySQL5.7.20，将以上软件安装到服务器，并配置好环境变量，MySQL账户：root，密码：1234
